Pelin ideana on rummuttaa musiikin tahtiin. Oikeasta rytmistä kertyy pisteitä.
Pelaaja voi valita useasta kappaleesta haluamansa ja katsoa kappale kohtaisesti
omat huippupisteensä. Pelissä voisi mahdollisesti vaikka jakaa huipputuloksensa
kaverin peliprofiiliin, facebookkiin tai sähköpostiin.

Kännykkää heiluttamalla käytetään shuffle toimintoa, jolloin peli valitsee satunnaisen
kappaleen pelattavaksi.
